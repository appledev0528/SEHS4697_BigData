📐 Key Formulas

MSE = sum((y - y_hat)^2) / n
RMSE = sqrt(MSE)
Accuracy = (TP + TN) / Total
Sensitivity = TP / (TP + FN) [Recall]
Specificity = TN / (TN + FP)
Precision = TP / (TP + FP)
Gain = CumYes / TotalYes  |  Lift = Gain / Random
